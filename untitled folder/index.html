What OK? does

OK? lets you set a daily reminder time to check in. If you don’t confirm your check-in, the app can send a pre-written alert email to a contact you choose.

  • Daily notification reminders
  • Optional contact email for alerts
  • Optional location sharing in an alert (only if enabled)

This website provides policy and support information for the OK? iOS app.
